## How to Use

1. Open `index.html` in any modern browser (Chrome, Safari, Edge)
2. Type or paste your text in the text area
3. Select a voice and adjust settings as needed
4. Click "Speak" to hear the text

## Controls

- **Text Input**: Enter your text here
- **Voice**: Choose from available system voices
- **Speed**: Adjust speech rate (0.5x - 2x)
- **Pitch**: Change voice pitch (0.5x - 2x)
- **Volume**: Control audio volume (0% - 100%)
- **Speak**: Convert text to speech
- **Stop**: Stop current speech
- **Clear**: Clear the text input

## Browser Support

Works in all modern browsers that support the Web Speech API:
- Chrome/Chromium
- Safari
- Microsoft Edge
- Firefox (limited support)

## Technology

- Pure HTML, CSS, and JavaScript
- Web Speech API
- No external dependencies
- Works completely offline
